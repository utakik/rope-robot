- # Swing Robot
ブランコロボット実験 / Swing robot experiments

## 目的
ブランコロボットの開発。  
ドラフト版成果品までの最短プロトタイピング演習。  
仮ログによるブランコの動きの確認、物理原則とログの関係を理解する。  

## 現状
- フォルダ作成完了 ✅
- 実験：ESP によるサーボ制御実験は、動きのラフな調整まで完了  
- 電源の安定化作業①完了
